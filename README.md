# GrabCab
GrabCab is a react native cross platform mobile app that enables user to login in into the app and request a taxi to their place of choice.

## How To Run
clone the repo using the link `https://github.com/MuthomiMate/GrabCab.git`
Once cloned cd into the root folder and run `yarn` to install the packages needed
To run the app use `expo start` command and you can use the available options to run using emulator or your device
